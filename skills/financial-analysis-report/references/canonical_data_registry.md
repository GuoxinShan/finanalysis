# Canonical Data Ownership

This file defines which financial metrics "belong" to which section. Workers must NOT restate metrics owned by other sections. If you need to mention a metric owned elsewhere, use a cross-reference like "(see Section V)" — never restate the numbers.

## Metric Ownership Table

| Metric | Owner Section | Do NOT repeat in |
|--------|--------------|-----------------|
| Revenue, growth % | V (Core Performance) | IV, VI, XII, XIII, XIV, XVI |
| Gross margin, gross profit | V (Core Performance) | IV, X, XI, XII |
| PBT, PAT, PATMI | V (Core Performance) | IV, XII, XIII |
| Attributable margin | V (Core Performance) | — |
| Admin expenses | XI (Expense Analysis) | IV, V, VI |
| Finance costs | XI (Expense Analysis) | V, IX, XIV, XVI |
| D/E ratio, gearing ratio | XIV (Solvency) | IX, XVII |
| Current ratio, quick ratio | XIV (Solvency) | IV, IX |
| Bank borrowings | XIV (Solvency) | IX, X, XVI, XVII |
| Operating cash flow (OCF) | XVI (Cash Flow) | IV, IX, XV |
| Free cash flow (FCF) | XVI (Cash Flow) | XIII |
| Order book | VI (Business Analysis) | VIII |
| Segment revenue/PBT | VI (Business Analysis) | V |
| NCI % of PAT | XII (Profitability) | V, XVII |
| ROE, ROA | XII (Profitability) | XIII |
| Risk matrix | IX (Risk Scan) | Standalone |
| Balance sheet summary | X (Major Items) | XVII |
| Expense breakdown | XI (Expense Analysis) | Standalone |
| Working capital metrics | XV (Operating Capability) | XIV, XVI |

## Cross-Reference Format

When you need to mention a metric owned by another section:

- "Revenue grew 58% (see Section V for full analysis)"
- "Leverage metrics are covered in Section XIV"
- "Cash flow details are in Section XVI"

Do NOT restate the numbers. Do NOT create a mini-table for metrics owned elsewhere.
