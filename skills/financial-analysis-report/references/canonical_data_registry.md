# Canonical Data Ownership

This file defines which financial metrics "belong" to which section. Workers must NOT restate metrics owned by other sections. If you need to mention a metric owned elsewhere, use a cross-reference like "(see Section III)" — never restate the numbers.

## Cross-Reference Format

When you need to mention a metric owned by another section:

- "Revenue grew 58% (see Section III for full analysis)"
- "Leverage metrics are covered in Section VII"
- "Cash flow details are in Section VIII"
- "Risk assessment in Section VI identifies..."

Do NOT restate the numbers. Do NOT create a mini-table for metrics owned elsewhere.
